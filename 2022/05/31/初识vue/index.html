# vue 的下载安装

# 官网下载 或 cdn 引入

# vue 的基础语法

# vue 的所有东西都是响应式的 改变 Vue 实例中 data 的值 view 中的值也会响应的改变
# 此处需要注意的是 当 data 中的属性在实例被创建的时候就已经存在 才能作出响应， 如果在实例创建完成之后 data 中才设置值 此时不会生效，如果预先知道值，可以先在 data 中设置属性值为空。
# 如果想解绑 ， 只加载一次 不会响应式改变 可以通过 Object.freeze (obj) 来设置

# v-bind

# v-bind 标签可以用来绑定标签的任一属性值 简写为 : 例如 :href

# v-if

此处注意 v-if 是是否加载标签 v-show 会加载标签 控制是否显示

# v-for

# items 来自于 Vue 实例中 data 中的值 items 可以为对象数组
# v-for 中也可以获取到 index 的值

# v-on

# v-on 监听用户的输入操作 clike dbclick ...

# 此处如果在 methods 的方法中改变了 data 中数据的值 页面上的值也会随之改变
# v-on 的缩写为 @ v-on:click 可以写为 @click

# Vue.component

# 自定义组件，理解为自定义标签 ， 将一块标签代码 封装起来

<div>
    <thisisthecomponentname></thisisthecomponentname>
</div>

Vue.component('this is the component name' ,{
	props : ['params'],
	template : '<p>this is custom tag<p>',
    methods : {
        
    }
})
# 组件中可以定义 参数 ，也就是标签的属性 ，可以通过 v-bind 绑定标签的属性 将值传入组件中 ， template 为标签模板 ，可以用到组件的属性 和 组件的方法，但是不能直接使用 Vue 实例的属性和方法，methods 为组件的方法。

# 生命周期

  1. beforeCreated
  2. created
  3. beforeMount
  4. mounted
  5. beforeUpdate updated
  6. beforeDestory destory

# v-mode

# v-mode 可以进行表单元素的双向绑定

# 表单中数据改变时，会绑定到 Vue 组件 data 中的数据也实时变更。

# slot

# 插槽 slot 即插即用 相当于在模板中先预留一部分组件的位置

# 普通 自定义组件中间是不可以加入标签的，可以在模板标签中预留插槽，就相当于预留了几个接口 ， 然后再使用该模板的时候 ，在模板标签的中间插入插槽模块，相当于实现了预留的接口。

<skill-list>
    <!-- 
这里的slot和 slot组件中的名称匹配 匹配到就放在插槽的位置
-->
    <skill-list-title slot="skill-title" :title='title'></skill-list-title>
    <!-- 
注意 ： 这里没有匹配到的标签 如果放在组件中 是不会生效的
-->
    <skill-list-title :title='title'></skill-list-title>
    <p> 这是一段文字 </p>
    <skill-list-arr slot="skill-arr" v-for='skill in skillList' :item='skill'>
    </skill-list-arr>
</skill-list>

--- 
<script>
    // 一个技能列表的组件 该组件有两个插槽 可以有标题 list中可以有技能列表 也可以没有
    Vue.component('skill-list',{
        template : '<div>\
                    <slot name="skill-title"></slot>\
                    <ul>\
                    <slot name="skill-arr"></slot>\
                    </ul>\
                    </div>'
    });
    // 定义一个title插槽组件
    Vue.component('skill-list-title',{
        props : ['title'],
        template : '<p>{{title}}</p>'
    })
    // 定义一个list插槽组件
    Vue.component('skill-list-arr',{
        props : ['item'],
        template : '<li>{{item}}</li>'
    })
    var vm = new Vue({
        el : '#app',
        data : {
            title : '技术列表',
            skillList : ['java','mysql','python']
        }
    })
</script>

# $emit

# $emit 为自定义事件 因为模板中的标签无法使用 Vue 中的属性和方法，因此通过 props 来传递 属性 ，通过 $emit 来传递方法

# 其实就是通过 $emit 定义一个自定义事件供 view 使用 ， view 通过该自定义事件调用 Vue 实例中的方法。
